# grind75-rs

Practicing using Rust using the 75 most common interview problems

## Usage

Each problem's main function asserts that the test cases provided in the problem description hold true. There is no output.

```bash
cargo run --example <PROBLEM_NAME>
```

## Problems

1. Two Sum [problem](https://leetcode.com/problems/two-sum) ([solution](examples/two_sum.rs))
2. Valid Parentheses [problem](https://leetcode.com/problems/valid-parentheses) ([solution](examples/valid_parentheses.rs))
3. Merge Two Sorted Lists [problem](https://leetcode.com/problems/merge-two-sorted-lists) ([solution](examples/merge_two_sorted_lists.rs))
